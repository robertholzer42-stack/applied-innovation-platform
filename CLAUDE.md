# CLAUDE.md - Resilient Innovation Agent Platform

## Project Overview

AI-powered multi-dimensional innovation practice. 13 specialized agents orchestrated by a Conductor to deliver Future Thinking + Design Thinking + Systems Thinking analysis for enterprise clients.

**Owner:** Bob Holzer, Next Horizon Innovations LLC
**GitHub:** robertholzer42-stack/resilient-innovation-platform

## Architecture

13 agents across 5 tiers:
- **Tier 1 (Client):** Navigator (intake), Publisher (deliverables)
- **Tier 2 (Core):** Scout (future, IFTF-aligned), Empathy (design), Architect (systems, DOC framework)
- **Tier 3 (Intersection):** Visionary (future+design), Integrator (design+systems), Sentinel (future+systems)
- **Tier 4 (Ops):** Radar (competitive), Banker (portfolio), Scorekeeper (metrics/DVFA), Bridge (change)
- **Tier 5 (Orchestration):** Conductor

## How It Works

Each agent is a SKILL.md file with personality, framework, and output format. The Conductor routes challenges to agents and synthesizes results. Engagement pipeline:

Navigator (intake) > Scout + Empathy + Architect (parallel) > Visionary + Integrator + Sentinel (synthesis) > Radar + Banker + Scorekeeper + Bridge (ops) > Conductor (synthesis) > Publisher (deliverables)

## Writing Rules

- Conversational and direct. Write like a smart colleague.
- Lead with the insight, not the process
- Every claim needs evidence
- No banned words: delve, landscape, synergy, leverage (as verb), robust, streamline, cutting-edge, paradigm, holistic, utilize
- No em dashes. Use commas, periods, or hyphens.
